- 创建文件
  - vim
  - mv
  - cp
- 创建目录
  - mkdir [option] dir-name
- 查看所有文件（包括隐藏文件）
　ls -a
- 查看目录树
  - tree -a 显示所有
  - tree -d 仅显示目录
  - tree -L n n代表数字,表示要显示多少层(L必须大写)
- MD5
  md3sum file
- 删除文件
  rm [option] file
  如果没有-r选项,rm不会删除目录
  -f 忽略不存在的文件,从不给出提示
  -r 指示rm将参数中列出的全部目录和子目录均递归地删除
  -i 进行交互式删除
